Code Execution

Run the flex code using flex command 
$ flex get_tokens.l 

Now a file lex.yy.c will be generated. Run generated c code using the command 
$ gcc lex.yy.c -ll 

Give filename as a argument to a.out 
$./a.out Sampleprogram.c 

An output file named flex_output.txt is generated which has tokens followed by token values.
